[asimov.js](https://www.npmjs.org/package/asimov.js) is a tool to help build websites and apps, using javascript and textfiles.

It features a simple and expressive syntax for creating content and layouts, simple folder structures for urls and sorting, a fast live-reloading development server and an even faster, clustered production server, built to handle massive amounts of requests.

## Learn how it works

If you are totally new to asimov.js, start with {{link "/docs/getting-started" "the basics"}} and the [Hello World](/blog/hello-world) blog post.

To learn more about what the CLI can do for you, check out {{link "/docs/commands" "Commands"}}.

Or if you want to dive right into creating creating content and designing pages, start with {{link "/docs/pages-and-urls" "Pages and Urls"}} and follow with {{link "/docs/templates-and-helper" "Templates and Helpers"}}.

If you want to keep up to date with all the new stuff in every version of asimov.js, keep checking out the {{link "/docs/changelog" "changelog"}} and {{link "/blog" "blog"}}.
