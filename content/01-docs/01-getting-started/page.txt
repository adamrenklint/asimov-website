So you've looked around, kicked the tires and you're now ready for a test run? Great, installing asimov.js and creating your new projects only takes a few seconds.

These instructions will fork the default project template and give you a great starting point for your next website and app project.

## Install the command line interface

    $ npm install -g asimov.js

## Create and start a new project

    $ ajs create myAwesomeProject
    $ cd myAwesomeProject
    $ ajs start
